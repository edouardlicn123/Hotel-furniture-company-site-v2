# 酒店家具企业官网模板（Flask + Python）

一套专为**酒店家具/商用家具**公司设计的现代化企业官网解决方案  
基于 Python + Flask 框架开发，**前后端分离**，简单易二次开发  
通过简单修改提示词 + 少量调整，即可快速改造成**其他行业**的企业官网

## 核心定位

给**中小型外贸/出口型企业**使用的最佳轻量级官网方案  
**前端全英文**（面向全球客户）  
**后台管理全中文**（老板/运营人员用得爽）

## 特色功能亮点

- 前后端分离架构，便于前端美工独立调整样式与页面
- **独家「官网 ↔ 画册」一键切换模式**  
  → 正常对外官网模式  
  → 切换成纯产品展示画册模式（可直接给供应商、经销商看货用）
- **分层式主题系统**
  → 支持用 AI（Grok/Claude/ChatGPT 等）快速生成并接入全新主题
- 完整的 **SEO 优化后台**  
  标题 / 描述 / 关键词 / OpenGraph / Twitter Card / 百度/谷歌站长代码 等一应俱全
- **产品「普通」+「精选」双分类体系**  
  精选可作为专题合集、爆款推荐、主推系列使用
- **客户询价一键生成邮件**（支持 SMTP）  
  客户选好产品 → 一键打包发邮件给业务员（支持附件图片）
- **深度适配 Grok 持续迭代开发**  
  提供清晰的提示词模板 + 模块化代码结构，后续想加功能找 Grok 改起来最快
- **真正的一键启动**  
  运行 `run.sh`
  → 自动检测/创建虚拟环境 → 安装依赖 → 初始化数据库 → 启动服务

## 快速上手

```bash
# Linux / macOS
chmod +x run.sh
./run.sh
```
