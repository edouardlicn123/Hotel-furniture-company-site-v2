# 用于酒店家具公司网站的模板

用于酒店家具公司的官方网站的模板，python+flask框架。满足一般中小型企业的官网需求，通过AI可快速修改部署为其他行业的企业网站。

本系统主要是给对外企业使用，所以前端英语后端中文。

# 特色功能

前端后端分离设计
特有官网/画册切换模式，后台一键切换。可将网站提供给供应商作为展示系统之用。
分层式CSS基础上的主题切换系统，自带多款主题，且可以用AI轻松制作新的主题
SEO，基本介绍等在后台已经做好设置功能，随时可以更改。
支持SMTP，客户可以将所需产品以邮件形式发给企业。
自带产品和精选两个类型，精选可作为产品的合集进行推广。
适配grok，有完善的与grok持续沟通后续开发的机制。
一键运行，自动配置好环境。


# 如何启动

执行run.sh（linux）
